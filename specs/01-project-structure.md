## Component Ownership

| Directory | Language | Purpose |
|:---|:---|:---|
| `cmd/nano-guard/` | Go | Binary entrypoint, wires up internal packages |
| `internal/hook/` | Go | Parses stdin JSON from IDE |
| `internal/git/` | Go | Runs `git diff`, handles non-git fallback |
| `internal/ollama/` | Go | HTTP client for Ollama `/api/generate` |
| `internal/evaluator/` | Go | Parses LLM JSON, maps to exit code |
| `internal/config/` | Go | Loads `nano-guard.config.json` |
| `testdata/` | JSON / diff | Fixtures for unit & integration tests |
| `e2e/` | Go | Full binary invocation tests (requires Ollama) |
| `prompts/` | Plain text | System prompt, loaded at runtime for easy editing |
| `init/src/` | TypeScript | `npx nano-guard init` bootstrapper |
| `init/src/__tests__/` | TypeScript | Unit tests for init CLI logic |
| `.github/workflows/` | YAML | CI pipeline вЂ” lint, unit test, type-check |
